    Open Nebula Python Bindings

    OpenNebula is a fantastic cloud management platform that incorporates a key feature: simplicity.

    OpenNebula seems the ideal cloud engine for Privazio. In fact I was very happy to see that there is a similar project already out there: clox.cloud that can run OpenNebula on two Raspbery Pi 3 using lightweight virtualization from LXD.

    I want to use OpenNebula as the Cloud Management platform for Privazio, however, I also want to automate its configuration with Ansible. Ansible plugins will therefore be required.

    OpenNebula is very actively been developed, and as such APIs do change. However, OpenNebula XML-RPC API includes XSD specifications.

    Pyone has been developed with maintainability in mind, it uses proxy objects and binding generators so that it will be easy to maintain as OpenNebula further evolves.

    Pyone will be used to implement Ansible modules for cloud management.

    Checkout Pyone in Github!
